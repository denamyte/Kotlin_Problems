Create a class named Student with two properties:

  • name of type String.
  • grade of type Int.

The name property should be a simple string variable, but the grade property should be a delegated property.

The delegate class should ensure that the grade property only accepts values between $0$ and $100$. If a value outside of this range is provided, the grade should default to $0$ for negative values and $100$ for values greater than $100$.

Additionally, every time a grade is set, print a message to the console indicating the new grade.

All packages you need to solve this challenge are already imported.

Sample Input:
110

Sample Output:
The new grade is 100



Memory limit: 256 MB
Time limit: 8 seconds
