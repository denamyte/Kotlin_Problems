Create a class named TextFormatter with a single property: text of type String. The text property should be a delegated property.

The delegate class should ensure that the text property always removes all the digits from the input and converts the remaining string to lowercase. Additionally, every time the text is set, print a message to the console indicating the new text.


Sample Input:
Remember, remember, the 5th of month 11

Sample Output:
remember, remember, the th of month



Memory limit: 256 MB
Time limit: 8 seconds
