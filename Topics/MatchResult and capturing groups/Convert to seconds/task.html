You have the following Kotlin program that reads a line of input consisting of multiple time durations separated by one space. Each duration follows the format: hh:mm:ss.

fun main() {
    val regex = """(\d{2}):(\d{2}):(\d{2})""".toRegex()
    val durations = readln().split(" ")

    convertToSeconds(durations, regex)
}

Write the function named convertToSeconds that converts each duration to seconds and prints each one on a new line.

Note that all input will be in the correct format.


Sample Input:
05:03:40 01:12:31 03:07:15

Sample Output:
18220
4351
11235



Memory limit: 256 MB
Time limit: 8 seconds
